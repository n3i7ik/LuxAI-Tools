üéØ Getting Started

1

Start Backend

Open terminal, navigate to backend folder, create virtual environment, install dependencies, run Django server

cd backend && python manage.py runserver
2

Start Frontend

Open another terminal, navigate to frontend folder, install npm packages, start Next.js development server

cd frontend && npm run dev
3

Start Redis

Open third terminal and start Redis server for caching

redis-server
4

Visit Application

Open browser and navigate to http://localhost:3000 to see your luxury AI suite

http://localhost:3000
